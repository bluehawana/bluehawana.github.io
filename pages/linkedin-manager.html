Usage Instructions

  1. Find your LinkedIn post: Go to your LinkedIn profile and find the post you want to sync
  2. Copy the URL: Click on the post date/time to get the full URL
  3. Paste and submit: Use the form above to add the post to your blog
  4. Edit if needed: The blog post will be created in the _posts directory

API Status

LinkedIn's API has restrictions on reading user posts. This tool uses manual URL submission as the primary method.
