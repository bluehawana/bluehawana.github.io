New Year, New Project: No.5 — Claude Code Proxy with NVIDIA Models

What if cost wasn't the barrier to exploring the world's most advanced AI models?
That's what inspired me to launch Project No.5. After completing projects 1-4, I wanted to take on a bigger challenge: making high-end #AI accessible to anyone curious enough to build.
Most developers never get to test top-tier models because they're too "expensive". Comparing them? Even harder. So people stick with familiar tools like #Copilot, missing out on models that could be faster, sharper, or better suited for specific tasks.
The game needs to be changed now.
Introducing the #NVIDIANIMProxySwitch —a tool that lets you switch between 182 NVIDIA NIM models in under a second. No restarts, no config edits, no hassle.
Here's what it offers:
- Instant model switching (under 1 second)
- 182 models ready to explore
- Web UI and REST API
- Zero downtime
- Seamless integration with Claude Code
- Fully open source
Built entirely with Qwen Coder via this project and Kiro, this project is AI helping build AI infrastructure.
And it’s completely free to use with NVIDIA’s NIM API, so you never have to choose between your budget and your curiosity.
Want to give it a try? Here is the demo for 7 days : models.bluehawana.com
Would like to have a deep understanding, you can find the repo in the article below. 
Feel free to star it, save it, or open a PR if you’d like to contribute.
I'm always excited to see what people create when limits disappear.
